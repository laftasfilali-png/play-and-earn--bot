# 🎮 Play & Earn Telegram Bot

بوت تيليغرام بسيط فيه لعبة Spin 🎡 ونظام نقاط (Coins) مع إمكانية تطويره إلى نظام VIP.

---

## 🔥 الميزات

- `/start` → عرض معلومات اللاعب (Coins, Spins, VIP)
- 🎡 Spin Button → لعب مباشر عبر زر (بدون كتابة)
- نظام Spins مجاني (3 مرات)
- نظام Coins (تجميع النقاط)
- نظام VIP (قابل للتطوير)

---

## 💻 التقنيات المستعملة

- Python
- python-telegram-bot
- Replit
